## `PMT.csv`
30个PMT探测器的位置信息，单位默认毫米 (如下所示)

| PMT_id | x | y | z |
| ------ | - | - | - |
| 0 | 428 | 139 | 700 |
| 1 | 265 | -364 | 700 |
| 2 | -245 | 756 | 245 |
| ... | ... | ... | ... |
| 29 | 245 | -756 | -245 |



数据来源于`PMT_Position.txt`，采用如下正则表达式修改：

    ```
    FROM: ^(\d+)\s+(-?\d+)\s+(-?\d+)\s+(-?\d+)$
    TO:   \1, \2, \3, \4
    ```
